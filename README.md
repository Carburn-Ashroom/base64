# Base64加解密工具

#### 介绍
用于加解密Base64编码的数据

#### 软件架构
使用**C++**的**Qt6.8**

#### 使用说明
开启**多行**功能时，**每一行原文**都会被翻译为一个单独的Base64串，反过来也可以同时翻译**多个Base64串**

[github仓库](https://github.com/Carburn-Ashroom/base64.git)
